# elena-email-psd
Practicing converting free PSD file to HTML file, BEM-methodology and RWD.

## Technologies I used in project
* webpack
* html
* flexbox
* sass
* yarn

### Install dependencies
```bash
$ yarn
```

### Build
```bash
$ yarn build
```

All builded files will be in `dist` directory.

### Run in dev mode
```bash
$ yarn dev
```
